# Features Roadmap - PronoHub

## Fonctionnalités futures

### Stats des équipes
Afficher les statistiques des équipes pour aider les utilisateurs à pronostiquer :
- Classement actuel de la compétition
- Nombre de victoires / nuls / défaites
- Performance domicile vs extérieur
- Buts marqués / encaissés
- Forme récente (5 derniers matchs)
- Confrontations directes (historique H2H)

**Monétisation** : Option payante ?

---

### Tendance des pronostiqueurs
Afficher les tendances de pronostics des autres participants (après clôture) :
- Pourcentage de votes pour chaque résultat (V1 / N / V2)
- Score le plus pronostiqué
- Consensus ou divergence sur un match

**Monétisation** : Option payante ?

---

## Notes
- Ces features nécessitent des appels API supplémentaires (football-data.org ou autre source)
- Évaluer l'impact sur les quotas API avant implémentation
- Considérer le cache pour limiter les appels
